## 문제발생 : 객체안 배열에 대해서  count 값만 수정을 하려고 했지만 수정이 안되는 문제가 발생
### 문제해결 : 예를 들어 a = ['ab', 'cd] 가 있다면 나는 b를 e로 수정을 하고싶었다. 그래서 a[0].split()를 통해 'ab'를 ['a', 'b']로 배열로 만들어서 a[0].split()[1] = "e" 이런식으로 변경을 했다. 당연하게 수정은 안되었고 간단하게 결론적으로는 a[0] = ae 이런식으로 수정을 하였다.
